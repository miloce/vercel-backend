健康管理小程序 API 服务

✅ API 服务运行正常

用户数据管理

  • GET /api/getUserData
    获取用户个性化数据
    userId string 必填
    用户唯一标识
    GET /api/getUserData?userId=user123 Response: { "success": true, "data": { "userId": "user123", "nickname": "张三", "gender": "male", "age": 28, "height": 175, "targetWeight": 70 } }
  • POST /api/saveUserData
    保存用户个性化数据
    userId string 必填
    用户唯一标识
    userData object 必填
    用户数据对象
    POST /api/saveUserData Content-Type: application/json { "userId": "user123", "userData": { "nickname": "张三", "gender": "male", "age": 28, "height": 175, "targetWeight": 70 } }

健康记录管理

  • GET /api/getHealthRecord
    获取指定日期的健康记录
    userId string 必填
    用户唯一标识
    date string 必填
    记录日期 (YYYY-MM-DD)
  • POST /api/saveHealthRecord
    保存用户的每日健康记录
    userId string 必填
    用户唯一标识
    date string 必填
    记录日期 (YYYY-MM-DD)
    weight number
    体重数据
    meals array
    饮食记录数组
    exercise object
    运动记录

身体数据统计

  • GET /api/getBodyStats
    获取用户的身体指标数据
    userId string 必填
    用户唯一标识
    date string
    记录日期 (YYYY-MM-DD)
    startDate string
    开始日期 (YYYY-MM-DD)
    endDate string
    结束日期 (YYYY-MM-DD)
  • POST /api/saveBodyStats
    保存用户的身体指标数据
    userId string 必填
    用户唯一标识
    date string 必填
    记录日期 (YYYY-MM-DD)
    weight number
    体重
    bodyFatPercentage number
    体脂率
    exerciseDuration number
    运动时长

微信认证

  • POST /api/getOpenId
    获取微信用户的OpenID
    code string 必填
    微信登录临时凭证
    POST /api/getOpenId Content-Type: application/json { "code": "wx_code_123456789" } Response: { "success": true, "data": { "openid": "wx_openid_123456789" } }
