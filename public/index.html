健康管理小程序 API 服务

✅ API 服务运行正常

用户数据管理

  • GET /api/getUserData
    获取用户个性化数据
    userId string 必填
    用户唯一标识
    GET /api/getUserData?userId=user123 Response: { "success": true, "data": { "userId": "user123", "nickname": "张三", "gender": "male", "age": 28, "height": 175, "targetWeight": 70 } }
  • POST /api/saveUserData
    保存用户个性化数据
    userId string 必填
    用户唯一标识
    userData object 必填
    用户数据对象
    POST /api/saveUserData Content-Type: application/json { "userId": "user123", "userData": { "nickname": "张三", "gender": "male", "age": 28, "height": 175, "targetWeight": 70 } }

健康记录管理

  • GET /api/getHealthRecord
    获取指定日期的健康记录
    userId string 必填
    用户唯一标识
    date string 必填
    记录日期 (YYYY-MM-DD)
  • POST /api/saveHealthRecord
    保存用户的每日健康记录
    userId string 必填
    用户唯一标识
    date string 必填
    记录日期 (YYYY-MM-DD)
    weight number
    体重数据
    meals array
    饮食记录数组
    exercise object
    运动记录

跑步记录管理

  • GET /api/getRunRecords
    获取用户的跑步记录
    userId string 必填
    用户唯一标识
    runId string
    跑步记录ID（不填则获取列表）
    limit number
    每页记录数量（默认10）
    skip number
    跳过的记录数量（分页用）
    GET /api/getRunRecords?userId=user123&limit=10&skip=0 Response: { "success": true, "data": { "records": [ { "runId": "run_123456", "date": "2023-04-05", "time": "00:30:15", "distance": 3.5, "speed": "8'35\"", "calories": 280, "runType": 0 }, // 更多记录... ], "total": 25, "page": 1, "limit": 10 } }
  • POST /api/saveRunRecord
    保存用户的跑步记录
    userId string 必填
    用户唯一标识
    runId string 必填
    跑步记录ID
    date string
    记录日期 (YYYY-MM-DD)
    time string
    运动时间 (HH:MM:SS)
    distance number
    跑步距离（公里）
    speed string
    平均配速
    calories number
    消耗卡路里
    runType number
    跑步类型（0:户外跑, 1:室内跑）
    points array
    轨迹点数组（仅户外跑）
    POST /api/saveRunRecord Content-Type: application/json { "userId": "user123", "runId": "run_123456", "date": "2023-04-05", "time": "00:30:15", "distance": 3.5, "speed": "8'35\"", "calories": 280, "runType": 0, "points": [ {"latitude": 39.9042, "longitude": 116.4074, "timestamp": 1617606000}, {"latitude": 39.9043, "longitude": 116.4075, "timestamp": 1617606010} ] }
  • POST /api/shareRunRecord
    分享跑步记录并更新分享计数
    userId string 必填
    用户唯一标识
    runId string 必填
    跑步记录ID
    shareType string
    分享类型（如 'timeline', 'werun', 'appMessage' 等）
    POST /api/shareRunRecord Content-Type: application/json { "userId": "user123", "runId": "run_123456", "shareType": "timeline" } Response: { "success": true, "message": "Share record updated successfully", "data": { "record": { // 跑步记录详情 }, "shareData": { "title": "我完成了3.5公里跑步，消耗了280卡路里！", "path": "/pages/run-share/run-share?runId=run_123456&userId=user123", "imageUrl": "https://static.luozhinet.com/xcx/assets/icons/share-run.png", "shareType": "timeline" } } }

微信认证

  • POST /api/getOpenId
    获取微信用户的OpenID
    code string 必填
    微信登录临时凭证
    POST /api/getOpenId Content-Type: application/json { "code": "wx_code_123456789" } Response: { "success": true, "data": { "openid": "wx_openid_123456789" } }
  • POST /api/getWeRunData
    获取微信运动步数数据
    code string 必填
    微信登录临时凭证
    encryptedData string 必填
    微信步数加密数据
    iv string 必填
    加密算法的初始向量
    POST /api/getWeRunData Content-Type: application/json { "code": "wx_code_123456789", "encryptedData": "encrypted_data_string", "iv": "iv_string" } Response: { "success": true, "data": { "stepInfoList": [ { "timestamp": 1617235200, "step": 6789 }, { "timestamp": 1617148800, "step": 8123 } ] }, "openid": "wx_openid_123456789" }
  • POST /api/getWxacode
    获取带参数的小程序码
    scene string 必填
    场景值参数，用于传递数据，如 "id=123"
    page string
    小程序页面路径，默认为 'pages/run/statement'
    POST /api/getWxacode Content-Type: application/json { "scene": "id=123456", "page": "pages/run/statement" } Response: 小程序码图片数据 (image/png)
